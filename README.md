# Multimodal Sentiment Analysis with Hybrid MCDM–Neural Fusion
This work presents an interpretable multimodal sentiment analysis framework that jointly exploits text and image information by combining multi-level sentiment cues with principled decision-making and deep learning. It extracts sentence- and segment-level sentiment from RoBERTa-based text encoders together with global, object-level, and scene-level sentiment from VGG16-based visual features, and organizes them into a structured 3 × 5 decision matrix for TOPSIS-based MCDM ranking of sentiment classes. By further integrating these MCDM scores with a conventional neural late-fusion of text and image classifiers via a tunable weighting scheme, the approach balances transparent, criteria-aware reasoning with data-driven predictive power, offering a flexible and extensible hybrid model for robust sentiment prediction across Positive, Negative, and Neutral classes.
